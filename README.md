# 蟒著喝咖啡 Busy Drinking Coffee ☕🐍

> **「今天不忙，只蟒著喝咖啡。」**
> 在喧囂、高速的都市人海中，給自己一處放空留白的溫柔角落。慢條斯理的溫度，治癒緊繃的身心。

![Design Preview](https://img.shields.io/badge/Design-Editorial_Minimalism-1C2D27?style=flat-square)
![TailwindCSS](https://img.shields.io/badge/CSS-TailwindCSS_3.x-AEEA00?style=flat-square&logo=tailwind-css&logoColor=1C2D27)

「蟒著喝咖啡」是一個結合**精緻餐飲**、**特殊寵物互動**與**生命教育**的非典型咖啡廳網頁。網站多數特寵大使來自棄養救援，我們期盼透過乾淨、精緻、充滿綠意的無壓環境，讓害羞溫柔的爬爬們化身親善大使，逐步消除大眾對爬蟲類的誤解與偏見。

---

## ✨ 網站亮點與核心特色

*   **社論雜誌風美學 (Editorial Minimalism)：** 運用大膽的留白、細緻的格線背景（Editorial Grid）與優雅的襯線字體（Cormorant Garamond），打造如同高級時尚雜誌般的視覺體驗。
*   **清新大地的色調調配：**
    *   `#1C2D27` (深林綠 Brand Dark) —— 沉穩的雨林與空間基調。
    *   `#AEEA00` (螢光青檸 Brand Lime) —— 亮眼的自然點綴與活力象徵。
    *   `#F3FAF5` (開心果綠 Brand Pistachio) —— 溫和無壓力的區塊底色。
*   **明星大使故事卡片：** 展示來自救援單位的球蟒（芭娜娜）、豹紋守宮（大橘）與睫角守宮（睫毛膏），並附帶專屬身世與性格介紹，強調生命教育。
*   **知識闢謠科學室：** 精選爬蟲類日常迷思解答（如鱗片觸感、自割機制），並提出飼養前的「靈魂三問」，引導大眾建立正確的飼育觀念。
*   **療癒感特調菜單：** 整合精心設計的主題餐點（如抹茶小苔蘚提拉米蘇、爬爬漸層氣泡特調）。
*   **完全響應式設計 (Responsive Design)：** 完美適應手機、平板與桌機，並包含流暢的平滑滾動（Smooth Scrolling）與行動版漢堡選單。

---

## 🛠️ 技術棧與外部資源

*   **網頁架構：** HTML5
*   **樣式框架：** [Tailwind CSS v3](https://tailwindcss.com/) (透過 CDN 引入並客製化 Theme 擴充顏色與字型)
*   **字型支援：** [Google Fonts](https://fonts.google.com/)
    *   英文字體：`Montserrat` (現代無襯線)、`Cormorant Garamond` (優雅古典襯線)
    *   中文字體：`Noto Sans TC` (思源黑體)
*   **圖標庫：** [Font Awesome 6.4.0](https://fontawesome.com/)
