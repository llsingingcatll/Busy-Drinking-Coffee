# 網站視覺風格規劃書：日系小清新風格

本規劃書旨在為 Hana 的個人作品集網站定義「日系小清新」的網站視覺視覺風格，確保整體網頁在視覺上呈現明亮、舒適、具備空氣感且細緻的氛圍，同時兼顧前端切版與使用者體驗。

## 一、 視覺核心概念 (Core Concept)
- **空氣感 (Airy)：** 留白大方，元件之間呼吸空間充足，避免視覺擁擠。
- **自然明亮 (Bright & Natural)：** 整體色調偏向明亮、低飽和度的自然光感。
- **細緻微互動 (Subtle Interactions)：** 動畫克制、輕盈（例如淡淡的淡入、優雅的 Hover 效果），展現前端技術的精緻度。

## 二、 色彩計畫 (Color Palette)
色彩選擇以低飽和度、高明度的日系色調為主，建立溫和、無壓力的瀏覽體驗：

| 角色 | HEX 色碼 | 視覺效果與建議用途 |
| :--- | :--- | :--- |
| **主背景色 (Background)** | `#F9F9F6` | 象牙白/質樸米白。比純白更溫和，減少長時間直視銀幕的疲勞感。 |
| **主文字色 (Main Text)** | `#3A3F44` | 石墨深灰。避免使用純黑（#000），使文字與背景的對比柔和、帶有印刷質感。 |
| **品牌主調色 (Primary)** | `#8FA499` | 鼠尾草綠/淡青。用於強調重要標題、導覽列點選狀態，象徵自然與清新。 |
| **輔助點綴色 (Accent)** | `#D4A373` | 溫潤淺木色/暖麥色。用於行動呼籲按鈕（CTA）或需要特別吸引注意的微小元件。 |

## 三、 字體與排版系統 (Typography)
- **中文字體優先順序：** `"PingFang TC"`, `"Noto Sans TC"`, `"Microsoft JhengHei"`, `sans-serif`。偏好俐落但有親和力的黑體。
- **英文字體建議：** `"DM Sans"` 或 `"Inter"`，線條幾何、簡潔，能很好地融入日系現代感。
- **字級與行距：** 內文字級建議為 `16px`，並嚴格維持 `1.65` 至 `1.8` 的行高（Line-height），確保段落呼吸空間。

## 四、 前端實作與 CSS 調校要點
- **陰影克制：** 避免使用重度、深色的陰影。若有卡片物件，使用非常淡的陰影：`box-shadow: 0 4px 20px rgba(0,0,0,0.03);`。
- **圓角設定：** 按鈕與卡片使用柔和的小圓角（如 `4px` 或 `6px`），既展現細緻感，又不失專業與俐落。
- **留白控制 (Padding/Margin)：** 各大區塊（Section）之間的垂直間距拉大（如 `80px - 120px`），創造大氣、不急躁的日系閱讀動線。
